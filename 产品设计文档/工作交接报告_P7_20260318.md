## 三、全局过滤 vs 策略配置（架构说明）

```
代币 → 🛡️全局过滤(硬底线) → 通过？→ 进入Watchlist → 🧪策略评分(6维) → 总分≥阈值？→ 买入信号 → 模拟交易
                ↓ 不通过                                              ↓ 不通过
              直接丢弃                                           继续观察
```

| | 全局过滤 🛡️ | 策略配置 🧪 |
|---|---|---|
| 作用 | 硬底线，不合格直接踢出 | 评分权重，打分排序 |
| 参数 | 链上持币人/币安KYC用户/流动性/市值/交易量/KOL/SM/Pro | 6维权重/买入阈值/止盈止损/超时 |
| 生效点 | Dashboard 热门代币 + Watchlist 入口 | Watchlist 内部评估 |
